# F1.1 Tool Rating
As a customer, I want to rate a tool (1-5 stars) so that others can benefit from my experience.

# Acceptance Criteria
- Rating submission only available after verified purchase
- Star selection interface (1-5 stars) clearly visible
- Optional text comment field (max 500 characters)
- Confirmation message displayed after successful submission
- Rating appears on product page within 24 hours
- User can edit their rating after submission
