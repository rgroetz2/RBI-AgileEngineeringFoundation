# US_1023 GDPR

AS a user
I WANT TO see a GDPR statement on the webiste 
SO I can see how my data is protected in the app


# Acceptance Criteria

A GDPR/privacy statement page exists and is publicly accessible without login.
A link to the statement is visible in the footer on every page of the website.
The statement covers: data collected, purpose of processing, user rights, and controller contact details.
The statement is written in plain, understandable language.
The statement page is responsive and displays correctly on desktop, tablet, and mobile.
