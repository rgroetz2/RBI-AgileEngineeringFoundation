## Document a bug
- Get a bug from the bug-list
- Analyze the bug and document it as GitHub Bug using following format:

  - Title:
    - ID3: Category Menu Chainsaw leads to 404
  - Environment:
    - AHDEV/http://109.176.250.87:4200/#/
  - Steps to Reproduce:
    - Open URL
    - Click “Categories”
    - Click “Chainsaws”
    - Expected Result:
      - Product overview page with chainsaws will be displayed
    - Actual Result:
      - Page with text “404 Error Page Not Found” is displayed
    - Visual Proof (screenshots, videos, text):

- Present it to the team


## Fix a bug
- Get another Github issue and create an automated test case with Playwright
- Fix the bug
- Test it with Playwright
- Release it
