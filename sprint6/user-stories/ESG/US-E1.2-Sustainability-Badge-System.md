# E1.2: Sustainability Badge System

As a customer
I want to see products marked with "Eco-Friendly" badges
So that I can quickly identify sustainable options

## Acceptance Criteria:
- Green "ECO" badge appears on products with CO₂ rating A or B
- Badge visible in product listing and detail pages
- "Show only eco-friendly products" filter option in sidebar
- Badge works on desktop and mobile interfaces
- Admin can activate/deactivate badge system
- Badge design follows existing UI patterns
