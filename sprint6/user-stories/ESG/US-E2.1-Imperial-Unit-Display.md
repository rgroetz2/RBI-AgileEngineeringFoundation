# E2.1: Imperial Unit Display
As a US contractor
I want to see tool dimensions in inches and pounds
So that I can quickly assess if the tool fits my needs without mental conversion
## Acceptance Criteria:
- All product dimensions display in imperial units (inches, feet)
- Weight displayed in pounds (lbs) and ounces (oz)
- Drill bit sizes in fractional inches (1/4", 3/8", 1/2")
- Power tool specs in US standards (120V, 15 Amp)
- Toggle between Imperial/Metric available
- Default to Imperial for US IP addresses
- Conversion accurate to 2 decimal places
