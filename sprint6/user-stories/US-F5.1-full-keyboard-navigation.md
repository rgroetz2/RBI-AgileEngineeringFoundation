# US-F5.1-full kexboard-navigation
As a user with motor impairments, 
I want full keyboard navigation 
so I can shop without a mouse.

# Acceptance Criteria
- All interactive elements accessible via Tab key
- Logical tab order follows visual layout
- Skip navigation link at page top
- Focus indicators clearly visible (2px solid outline)
- No keyboard traps (user can always navigate away)
- Dropdown menus accessible via Arrow
