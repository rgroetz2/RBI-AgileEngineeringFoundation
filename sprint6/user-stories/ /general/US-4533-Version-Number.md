# US-4533 Version Number

As a service desk agent, 
I want the build version of the web-shop to be displayed in the user interface, 
so that I can easily identify and troubleshoot issues reported by customers based on the specific version of the application they are using.



ACC1: New menu “Info” in main menu is available. Modal window (UI) will be displayed with following information:

- Build Number
- Date of build
- Angular Version
