# US-F5.3-high-color-contarast
As a user with visual impairment, 
I want high color contrast 
so I can read all text clearly.

# Acceptance Criteria 
- Text contrast ratio minimum 4.5:1 (normal text) / 3:1 (large text)
- Color not used as sole indicator (use icons + color)
- Interactive elements have minimum 44x44px touch target size
- Text resizable up to 200% without breaking layout
- No information loss when zooming to 400%
- Focus indicators meet 3:1 contrast ratio
