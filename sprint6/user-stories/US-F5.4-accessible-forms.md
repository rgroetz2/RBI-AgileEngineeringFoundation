# US-F5.4-accessible-forms 
 As a user with cognitive impairments, 
 I want accessible forms 
 so I can complete orders successfully.


# Acceptance criteria
- Form fields have clear, descriptive labels
- Required fields marked with asterisk and "required" text
- Input format examples shown (e.g., "MM/DD/YYYY")
- Error messages specific and actionable
- Errors announced to screen readers immediately
- Error summary at top of form with links to fields
- Timeout warnings with option to extend session
