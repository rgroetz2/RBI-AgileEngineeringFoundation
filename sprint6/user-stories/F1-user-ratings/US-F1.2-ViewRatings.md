# F1.2 View Ratings
As a visitor, I want to see average ratings on product pages so I can assess tool quality.

# Acceptance Criteria
- Average rating (stars) displayed next to product title
- Number of reviews shown in parentheses (e.g., "4.5 ★ (127 Reviews)")
- Click on rating scrolls to review section on page
- Rating distribution histogram visible (5-star, 4-star, etc.)
- If 0 reviews: Display "No reviews yet"
- Mobile-responsive display of rating information
