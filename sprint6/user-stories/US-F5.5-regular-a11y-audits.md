# US-F5.5-regular-a11y-audits
As an accessibility specialist, 
I want regular a11y audits 
so we ensure WCAG compliance.



# Acceptance Criteria
- Automated testing with aXe, Lighthouse, WAVE
- Manual testing with NVDA, JAWS, VoiceOver screen readers
- Keyboard-only navigation testing
- Color contrast analysis tools
- User testing with people with disabilities
- Accessibility statement page published
- WCAG 2.1 AA compliance badge displayed
