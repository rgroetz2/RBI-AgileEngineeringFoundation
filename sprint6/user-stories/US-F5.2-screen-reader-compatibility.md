 # US-F5.2-screen-reader-compatibility
   As a blind user, 
   I want screen reader compatibility 
   so I can shop independently.

# Acceptance Criteria
- All images have descriptive alt text
- Form fields have associated labels (not just placeholders)
- ARIA landmarks used (nav, main, aside, footer)
- ARIA live regions for dynamic content updates
- Proper heading hierarchy (h1, h2, h3)
- Link text is descriptive (not just "click here")
- Product prices and ratings announced correctly
