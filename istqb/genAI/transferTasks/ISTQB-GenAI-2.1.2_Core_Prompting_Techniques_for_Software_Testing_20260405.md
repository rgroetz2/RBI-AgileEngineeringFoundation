### 1. Title
Compare Core Prompting Techniques on One QA Problem

---

### 2. Syllabus Reference
ISTQB GenAI – 2.1.2 Core Prompting Techniques for Software Testing

---

### 3. Learning Objective
Evaluate how different prompting techniques affect output quality for test-related tasks.

---

### 4. Context / Scenario
The team is unsure when to use zero-shot, few-shot, and stepwise prompting. You will evaluate all three on the same testing challenge.

---

### 5. Task Instructions
1. Select one scenario (e.g., checkout validation tests).
2. Create 3 prompts using:
   - Zero-shot
   - Few-shot
   - Stepwise reasoning instruction
3. Run all prompts and collect outputs.
4. Score outputs for correctness, completeness, and execution readiness.
5. Recommend which technique fits which situation.

---

### 6. Expected Outcome / Deliverable
A comparison table and short technique-selection guideline.

---

### 7. Timebox
25 minutes

---

### 8. Hints (Optional)
- Keep scenario and constraints identical across all runs.
- Focus on output usability for real test execution.
