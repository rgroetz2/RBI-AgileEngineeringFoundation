### 1. Title
Generate a Reliable AI-Assisted Daily Test Status Briefing

---

### 2. Syllabus Reference
ISTQB GenAI – 2.2.4 Test Monitoring and Control with Generative AI

---

### 3. Learning Objective
Use GenAI to summarize test execution status while preserving factual accuracy and decision relevance.

---

### 4. Context / Scenario
Daily status updates are time-consuming. The team wants an AI-assisted summary format for progress, blockers, and risks.

---

### 5. Task Instructions
1. Collect one day of test execution data (passed, failed, blocked, open defects).
2. Prompt GenAI to produce a concise daily report.
3. Verify every AI claim against source data.
4. Correct inaccuracies and add missing risk notes.
5. Create a reusable reporting prompt template.

---

### 6. Expected Outcome / Deliverable
One validated daily report and one reusable prompt template.

---

### 7. Timebox
20 minutes

---

### 8. Hints (Optional)
- Ask for explicit metrics and trend statements.
- Keep human approval as mandatory step.
