### 1. Title
Control Non-Determinism in AI-Assisted Test Design

---

### 2. Syllabus Reference
ISTQB GenAI – 3.1.4 Handling Non-Deterministic Behavior

---

### 3. Learning Objective
Measure output variability and define controls for reproducible QA usage.

---

### 4. Context / Scenario
Running the same prompt multiple times gives different results, making review and audit difficult.

---

### 5. Task Instructions
1. Use one fixed prompt and run it 5 times.
2. Compare outputs for stable vs variable elements.
3. Identify where variability is acceptable vs risky.
4. Define controls: templates, constraints, fixed output schema, review checkpoints.
5. Re-run once with controls and record improvement.

---

### 6. Expected Outcome / Deliverable
Variability analysis and repeatability control guideline.

---

### 7. Timebox
25 minutes

---

### 8. Hints (Optional)
- Separate wording variability from semantic variability.
- Focus controls on decision-relevant content.
