### 1. Title
Select the Best-Fit Model Class for QA Use Cases

---

### 2. Syllabus Reference
ISTQB GenAI – 5.1.3 Selection of LLMs/SLMs

---

### 3. Learning Objective
Apply weighted decision criteria to select model options aligned with QA requirements and constraints.

---

### 4. Context / Scenario
You need to recommend model options for test tasks under quality, privacy, and cost constraints.

---

### 5. Task Instructions
1. Define weighted criteria: quality, cost, latency, privacy, controllability.
2. Compare at least 3 model options/classes.
3. Score each model and compute ranking.
4. Document assumptions and constraints.
5. Recommend one primary and one fallback option.

---

### 6. Expected Outcome / Deliverable
Weighted model selection matrix with recommendation.

---

### 7. Timebox
25 minutes

---

### 8. Hints (Optional)
- Avoid ranking without explicit weight rationale.
- Include one risk per selected option.
