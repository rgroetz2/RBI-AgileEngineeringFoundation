### 1. Title
Design a Reference Architecture for LLM-Based Test Support

---

### 2. Syllabus Reference
ISTQB GenAI – 4.1.1 Key Architecture Components

---

### 3. Learning Objective
Identify essential architecture components and control points for reliable AI-assisted testing.

---

### 4. Context / Scenario
Your team is planning an LLM-based testing setup and needs a minimal but robust architecture blueprint.

---

### 5. Task Instructions
1. Define core components: interface, model layer, context store, validation, logging, feedback loop.
2. Draw a simple flow (text-based) from input to approved output.
3. Mark control points for quality and security checks.
4. Identify 3 architecture risks and one mitigation each.

---

### 6. Expected Outcome / Deliverable
Architecture sketch with control points and risk notes.

---

### 7. Timebox
30 minutes

---

### 8. Hints (Optional)
- Keep the design implementation-agnostic.
- Prioritize traceability and reviewability.
