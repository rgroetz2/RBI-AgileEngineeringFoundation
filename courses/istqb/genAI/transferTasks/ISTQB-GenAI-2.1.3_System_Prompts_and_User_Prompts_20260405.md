### 1. Title
Build a Reusable System Prompt for QA Work

---

### 2. Syllabus Reference
ISTQB GenAI – 2.1.3 System Prompts and User Prompts

---

### 3. Learning Objective
Differentiate responsibilities of system and user prompts and implement a reusable prompt package.

---

### 4. Context / Scenario
Your team needs output consistency across different testers and use cases.

---

### 5. Task Instructions
1. Write one system prompt defining AI role, quality criteria, and prohibited behaviors.
2. Create 3 user prompts for:
   - Test analysis
   - Test design
   - Defect triage
3. Execute all 3 user prompts with the same system prompt.
4. Check whether output structure remains consistent.
5. Refine the system prompt once.

---

### 6. Expected Outcome / Deliverable
Prompt package with system prompt v1/v2 and 3 validated user prompts.

---

### 7. Timebox
20 minutes

---

### 8. Hints (Optional)
- Add explicit “if uncertain, ask clarifying questions” behavior.
- Define mandatory output sections.
