### 1. Title
Run a Structured 3-Cycle Prompt Refinement Loop

---

### 2. Syllabus Reference
ISTQB GenAI – 2.3.2 Techniques for Evaluation and Iterative Prompt Refinement

---

### 3. Learning Objective
Improve AI output quality through iterative evaluation and prompt refinement.

---

### 4. Context / Scenario
One-shot prompting produces mixed-quality test artifacts. You need an empirical refinement process.

---

### 5. Task Instructions
1. Choose one objective (e.g., generate high-quality negative checkout tests).
2. Execute prompt v1 and score output with defined metrics.
3. Refine prompt to v2 based on observed gaps; rescore.
4. Refine prompt to v3; rescore.
5. Summarize quality trend and final prompt guidance.

---

### 6. Expected Outcome / Deliverable
Iteration log (v1–v3), scores, and final recommended prompt.

---

### 7. Timebox
30 minutes

---

### 8. Hints (Optional)
- Change one major variable per iteration when possible.
- Keep test objective constant across all rounds.
