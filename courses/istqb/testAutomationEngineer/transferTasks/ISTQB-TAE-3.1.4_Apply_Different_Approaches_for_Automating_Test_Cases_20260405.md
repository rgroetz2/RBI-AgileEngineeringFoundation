### 1. Title
Compare Automation Approaches on the Same Toolshop Feature Set

---

### 2. Syllabus Reference
ISTQB TAE – 3.1.4 Apply Different Approaches for Automating Test Cases

---

### 3. Learning Objective
Select suitable automation approaches by comparing applicability, effort, and maintainability on real Toolshop tests.

---

### 4. Context / Scenario
The team debates whether to use only scripted tests or combine multiple automation approaches.

---

### 5. Task Instructions
1. Choose 3 Toolshop checks: product search, login validation, cart quantity update.
2. For each check, evaluate three approaches: data-driven, keyword-driven, and script-centric.
3. Document implementation sketch for each approach in 2–3 lines.
4. Rate each approach per check on initial effort and long-term maintenance (Low/Medium/High).
5. Recommend one primary approach per check with rationale.

---

### 6. Expected Outcome / Deliverable
A comparison table with approach ratings and per-check recommendations.

---

### 7. Timebox
30 minutes

---

### 8. Hints (Optional)
Use one approach by default, but allow exceptions for tests with heavy data variation.
